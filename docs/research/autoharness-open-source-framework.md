# AutoHarness Open Source Framework

Last updated: 2026-06-02

Primary source: [aiming-lab/AutoHarness](https://github.com/aiming-lab/AutoHarness)

## Category

Open-source harness-governance framework.

## Core Idea

This is an open-source framework called AutoHarness that wraps agent clients
with governance features such as tool governance, context management, risk
classification, permissions, output sanitization, audit logs, cost attribution,
and multi-agent profiles.

## Why It Matters Here

This is not the same as the Google DeepMind AutoHarness paper. It is still a
useful resource for practical harness-governance patterns.
