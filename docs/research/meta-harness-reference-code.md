## Core Idea

The Stanford IRIS Lab repository provides reference code for Meta-Harness,
including onboarding guidance and reference experiments for text classification
and Terminal-Bench 2 scaffold evolution.

## Why It Matters Here

This is useful as a practical implementation reference for applying a
Meta-Harness-style loop to a new domain.

## Implementation Notes

- Study the onboarding flow for generating a `domain_spec.md`.
- Compare their artifact layout to this repository's workspace layout.
- Look at how proposer interactions are wrapped and logged.
- Keep our implementation independent; use the repo as design reference, not as
  code to copy wholesale.
