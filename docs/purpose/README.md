## Short Version

HarnessGenerator treats AI system behavior as an engineering object that can be measured,
compared, and improved. Instead of changing prompts by intuition, it defines task workspaces,
contracts, harness strategies, eval datasets, metrics, safety constraints, and generated follow-up
tasks. The goal is to make harness improvement repeatable and auditable.
