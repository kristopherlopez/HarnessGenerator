# Intake

Use this folder for progressive source collection before examples become labelled dataset cases.

Recommended flow:

1. Add YouTube links to `youtube_links.jsonl`.
2. Add notes about expected speakers, useful timestamp ranges, and permissions.
3. Create a draft case under `datasets/drafts/cases/`.
4. Add local media or clips under `datasets/drafts/media/` when available.
5. Move the case into `datasets/small_gold/cases/` only after transcript segments and true speaker labels are reviewed.

`small_gold` should stay curated. Raw links and partial annotations belong here or in `datasets/drafts/`.
